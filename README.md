# APIs

## formatter
主要為文字格式化工具，包含常用的Regex以及一些與金額格式化相關的方法

## fragment
與`Fragment`	開發相關的方法，例如快速返回第一頁之類的

## html
`HtmlTextViewRender` : 專門用來處理用TextView渲染Html語法時的圖片載入問題 

## network
對`Okhttp`的再封裝

## recyclerview
一些在使用`RecyclerView`時常會使用到的物件

## other
### ImageLoaderUtil
封裝`ImageLoader`的初始化參數
### JSON
封裝`json`，處理`JsonException`問題
